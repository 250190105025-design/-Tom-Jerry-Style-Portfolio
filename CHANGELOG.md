## [1.0.0] — 2025-03-02 🎉 Initial Release

### ✨ Added
- Full Tom & Jerry cartoon-style portfolio — single `index.html` file
- **Hero section** with animated SVG Jerry character holding cheese
- **Running chase scene** — Jerry and Tom animate across the footer of hero
- **Tom's paw cursor** — custom SVG cursor with lag-follow effect
- **Cheese crumb trail** — random cheese particles follow the cursor
- **Frying pan 🍳 interaction** — click to BONK! Jerry changes expression
- **Cheese wheel 🧀 interaction** — click for cheese shower + confetti
- **Skill cards** (6 total) with POW/ZAP/BOOM comic bang click effects + ripple
- **Project cards** (6 total) with like button, ribbon labels, hover tilt
- **Experience timeline** with dot click interactions and Jerry quips
- **Contact section** with all social links
- **🎮 Chase Mini-Game** — steer Jerry with mouse, click cheese to score, rank system
- **Animated stat counters** — count up on scroll into view
- **Scrolling red tech marquee** banner
- **Scroll-reveal animations** on all cards and timeline items
- **Active nav tracking** — highlights current section on scroll
- **Speech bubble toast system** — Jerry quip notifications
- **Confetti system** — multi-color cartoon confetti bursts
- **BANG text effects** — comic POW/BONK/ZAP/WHAM/LOVE overlays
- **Konami Code easter egg** — ↑↑↓↓←→←→BA triggers mega confetti
- **Scroll-to-top button** — appears after scrolling 400px
- GitHub Actions workflow for auto-deploy to GitHub Pages
- MIT License

### 🎨 Design
- Hanna-Barbera color palette: Gold, Tomato Red, Cobalt Blue, Warm Brown, Cream
- Thick `4-5px` black outlines on all components (cartoon cell style)
- Hard box-shadows `5px 5px 0 #000` — zero soft blurs
- Halftone dot texture background overlay
- Titan One + Fredoka One display fonts
- Fully responsive layout (desktop, tablet, mobile)
